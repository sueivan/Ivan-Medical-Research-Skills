# 设计选择

| 设计 | 适合的问题 | 主要优势 | 关键风险 |
|---|---|---|---|
| 个体随机试验 | 可分配干预的因果效果 | 基线混杂平衡 | 分配隐藏、盲法、失访、依从、外部效度 |
| 集群随机试验 | 群体实施或污染明显 | 适合系统级干预 | 聚类数少、招募偏倚、分析单位错误 |
| 交叉试验 | 稳定慢性状态和可逆效应 | 个体内比较 | 延滞、周期、不可逆结局 |
| 阶梯楔形 | 分阶段系统实施 | 所有集群最终接受 | 时间趋势、复杂分析、实施混杂 |
| 前瞻性队列 | 暴露后结局、预后 | 时间顺序清楚、可测多结局 | 混杂、失访、暴露变化 |
| 回顾性队列 | 现有纵向数据 | 快速、规模大 | 数据为其他目的生成、误分类、缺失 |
| 病例对照 | 稀有结局/长潜伏期 | 效率高 | 来源人群、对照选择、回忆/测量偏倚 |
| 横断面 | 患病率和同期关联 | 快速 | 时间顺序不清、幸存者偏倚 |
| 诊断准确性 | 指数试验相对参考标准 | 直接估计敏感度/特异度等 | 谱偏倚、验证偏倚、阈值选择 |
| 准实验 | 随机化不可行的政策/系统效果 | 可利用外生变化 | 趋势、同期干预、模型假设 |
| 病例系列 | 新现象和信号描述 | 生成假设 | 无对照、选择性强，不能估计比较效应 |

## 选择顺序

1. 问题是否要求因果效应？
2. 干预是否可伦理且可行地随机分配？
3. 结局稀有还是暴露稀有？
4. 是否需要时间顺序或动态暴露？
5. 数据能否准确识别资格、时间零点、暴露和结局？
6. 是否存在政策阈值、分期实施或中断时间序列等准实验机会？

## 不应机械选择

- “罕见病一定病例对照”：若有完整人群数据库，队列也可能合适。
- “真实世界数据一定更真实”：数据适用性和偏倚结构决定可信度。
- “RCT 一定最佳”：问题、伦理、估计目标和外部效度决定适用性。
- “前瞻性一定优于回顾性”：核心是数据生成、测量、时间线和偏倚控制。
