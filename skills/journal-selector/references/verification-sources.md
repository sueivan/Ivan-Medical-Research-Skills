# 期刊信息核验来源

## 第一层：期刊与出版社官网

核对 Aims & Scope、文章类型、作者指南、同行评审、费用、OA/许可证、版权、数据/代码、预印本、AI、伦理、撤稿和联系方式。确认网址由真实出版社或学会控制，ISSN 和刊名一致。

## 第二层：权威目录

- Web of Science Master Journal List：核对具体索引及覆盖变化；JCR 指标另行核验。
- NLM Catalog：区分 Currently indexed for MEDLINE、PubMed 引用来源及其他状态。
- PubMed/PMC：文章可检索或全文存档不自动等于期刊 MEDLINE 收录。
- Scopus Sources：核对覆盖年份、状态和指标口径。
- DOAJ：用于完全开放获取期刊及其透明度信息；未收录不是定罪。
- ISSN Portal：核对正式刊名、ISSN 和出版者身份。
- 机构/基金名单：仅在用户评价、资助或 OA 合规需要时使用，并记录版本。

## 核验规则

1. 保存查询日期、页面、刊名、ISSN和状态原文。
2. 指标必须记录年份和口径；不同平台指标不可直接比较。
3. 发现官网与目录冲突时以“待核验/暂缓”处理，联系官方而非猜测。
4. 期刊更名、合并、转出版社或暂停收录时记录覆盖年份。
5. 第三方聚合网站只作为线索，不作为最终证据。

## 时效信息

审稿速度、接受率和 APC 变化快。优先官网公开统计，写明统计口径；作者经验和网络传言仅标记“非官方信息”，不可作为保证。
