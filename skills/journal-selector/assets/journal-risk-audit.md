# 期刊诚信风险审计

## 身份与透明度

- [ ] 官方域名、刊名、ISSN和出版社与权威目录一致
- [ ] 编辑部和编委姓名、单位可核验
- [ ] 同行评审流程、文章类型和编辑责任清楚
- [ ] 所有投稿至出版费用及豁免条件公开
- [ ] OA、许可证、版权和长期存档政策清楚
- [ ] 联系地址和邮箱与出版者身份一致

## 高风险信号

- [ ] 冒用或无法核验的影响指标/数据库收录
- [ ] 仿冒知名期刊网站或域名异常
- [ ] 保证录用、保证收录或极不现实的审稿承诺
- [ ] 大量与稿件无关、催促性或不专业邀稿
- [ ] 投稿后才披露费用或以撤稿费施压
- [ ] 编委否认任职、身份虚假或学科明显不匹配
- [ ] 期刊范围异常宽泛、特刊失控或同行评审描述空洞
- [ ] 已发表文章存在批量模板、伪造同行评审或论文工厂信号
- [ ] 撤稿、更正、投诉和研究诚信政策缺失或不可执行

## 判读

单一信号通常不足以定性。重大身份/费用/评审疑点未解决时标记“暂缓”，保存证据并请图书馆员、研究诚信办公室或出版专家复核。
