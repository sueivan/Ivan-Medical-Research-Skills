# 检索验证与审计

## 已知相关文献集

在写最终式前建立小型验证集，来源可包括关键综述纳入研究、指南、专家推荐和种子论文。记录文献是否被 Scopus 收录；数据库未收录与检索漏检必须分开。

## 漏检归因

对每篇未检出文献检查：

1. Scopus 是否收录该记录；
2. 题名、摘要、关键词是否完整；
3. 概念词是否存在于目标字段；
4. 短语、通配符或邻近是否过窄；
5. 日期、语言、文献/来源类型是否排除；
6. 是否存在拼写、旧称或缩写缺口。

只有与纳入标准一致且不会造成不合理噪声时才扩展术语。

## 精确度抽查

从不同年份、来源和排序区间抽样，避免只看前几条。分类非相关原因：概念多义、缩写歧义、上下文缺失、索引误配或错误限制。优先用字段和邻近改善，谨慎使用排除词。

## 增量测试

每次只改变一个元素，记录修改前后结果数、验证文献、随机样本精确度和意外丢失。保留版本号，不覆盖旧检索式。

## 同行评审重点

问题转换、概念遗漏、术语拼写、括号、字段、短语、截词、邻近、限制、排除、日期边界、可复现性和数据库转换。同行评审意见及处理结果留痕。
