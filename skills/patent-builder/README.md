# Patent Builder

Status: **operational invention-development beta (v0.1.0)**

`patent-builder` prepares biomedical and health-technology inventions for rigorous patent-attorney review by linking prior art, technical problem, distinguishing features, technical effects, enablement, embodiments, figures, and claim support.

## Main outputs

- confidential invention disclosure
- disclosure and ownership timeline
- prior-art search and comparison matrix
- feature–effect evidence map
- provisional novelty/inventive-step analysis
- claim tree and fallback positions
- embodiments and figure plan
- experiment/evidence-gap plan
- attorney-handoff package

## Invoke

```text
Use @patent-builder to structure this invention for attorney review. Protect confidentiality, verify prior art, define the technical contribution chain, separate completed from planned evidence, build claims and fallbacks, and flag jurisdiction-specific legal questions.
```

Patentability, freedom to operate, filing, prosecution, and legal conclusions require qualified patent counsel.
