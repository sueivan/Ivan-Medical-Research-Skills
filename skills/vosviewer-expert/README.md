# VOSviewer Expert

Status: **operational beta (v0.1.0)**

`vosviewer-expert` designs, audits, interprets, and reports reproducible VOSviewer bibliometric analyses.

## Main outputs

- corpus/import audit
- analysis and parameter protocol
- VOSviewer thesaurus and cleaning log
- item, link, cluster, overlay, and density interpretation tables
- parameter-sensitivity log
- publication-ready figure legends
- bounded methods and results text

## Invoke

```text
Use @vosviewer-expert to design and audit co-authorship, keyword co-occurrence, and co-citation maps for this WoS corpus. Justify counting, thresholds, normalization, clustering, thesaurus cleaning, overlay settings, and sensitivity analysis.
```

Use `wos-search-builder` first when corpus construction is unresolved.
