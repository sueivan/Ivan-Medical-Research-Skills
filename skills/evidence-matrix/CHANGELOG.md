# Changelog

## 0.1.0 — 2026-07-12

- Replaced the scaffold with an operational biomedical evidence-extraction workflow.
- Added study/report linkage, provenance states, outcome-level extraction, consistency checks, and missing-data controls.
- Added study, report, outcome, data-dictionary, and query-log templates.
- Added safeguards against invented data, double counting, and interpretation of nonsignificance as no effect.
