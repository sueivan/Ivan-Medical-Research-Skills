# Grant Builder

Status: **operational beta (v0.1.0)**

`grant-builder` develops evidence-based, call-compliant proposals for biomedical, public-health, nursing, medical-education, social-science, policy, and interdisciplinary funding.

## Main outputs

- call compliance matrix
- go/no-go fit assessment
- evidence-gap and innovation map
- specific aims and aim–method alignment
- study design, analysis, rigor, ethics, and governance plan
- work packages, milestones, timeline, budget, and risk register
- impact and knowledge-translation pathway
- reviewer-response matrix

## Invoke

```text
Use @grant-builder to assess and develop this proposal for [official funding call]. Verify the current call first, conduct a go/no-go fit assessment, lock aims before drafting, align methods and budget, and flag every unverified institutional commitment.
```

Current call documents and institutional rules must be verified before finalization.
