# Web of Science Search Builder Quality Checklist

## Scope
- [ ] Study purpose and corpus boundary are explicit
- [ ] Web of Science collection is named
- [ ] Included indexes/editions are recorded
- [ ] Timespan and search date are recorded
- [ ] Sensitivity/precision priority is stated

## Concepts and syntax
- [ ] Essential concept blocks are justified
- [ ] PubMed/MeSH terms were reassessed as free text
- [ ] Fields are chosen deliberately
- [ ] `TS=` implications, including Keywords Plus, are considered
- [ ] Boolean logic and parentheses are complete
- [ ] Wildcards are tested for semantic noise
- [ ] Quoted phrases are justified
- [ ] `NEAR/x` distances are justified and tested
- [ ] `NOT` is absent or conservatively tested

## Restrictions
- [ ] Years are explicit
- [ ] Document types are justified
- [ ] Language and other filters are recorded
- [ ] Early-access handling is stated
- [ ] Partial current-year treatment is stated

## Validation
- [ ] Each concept block was tested separately
- [ ] Combined blocks were tested sequentially
- [ ] Seed records were checked when available
- [ ] A precision sample was reviewed
- [ ] Keywords Plus/boundary records were inspected
- [ ] Earliest and latest years were inspected
- [ ] Counts are reported only if actually observed
- [ ] Exact Search History is saved or labeled unrun

## Export and completion
- [ ] Export content and format are recorded
- [ ] Batch ranges and sort order are recorded
- [ ] Export date and total records are recorded
- [ ] Intended analysis software/version is recorded
- [ ] Deduplication and cleaning handoff is stated
- [ ] Another researcher could reproduce the corpus
