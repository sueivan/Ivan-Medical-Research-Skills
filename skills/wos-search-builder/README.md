# Web of Science Search Builder

Status: **operational beta (v0.1.0)**

`wos-search-builder` creates and audits reproducible Web of Science Core Collection strategies for biomedical reviews and bibliometric studies.

## Main outputs

- collection/index scope statement
- corpus-boundary and concept table
- readable and single-line Advanced Search queries
- `TS=`, field, wildcard, phrase, and `NEAR/x` decisions
- staged-test and validation log
- exact Search History record
- reproducible export plan
- manuscript-ready methods paragraph

## Invoke

```text
Use @wos-search-builder to construct a Web of Science Core Collection strategy for [topic]. This is a bibliometric study using CiteSpace and VOSviewer. Define the corpus boundary, indexes, years, document types, validation checks, and export settings.
```

Do not paste a PubMed strategy unchanged; WoS requires database-specific translation.
