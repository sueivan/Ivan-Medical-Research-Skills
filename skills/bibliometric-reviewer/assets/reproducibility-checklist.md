# 可复现性检查清单

- [ ] 数据库、平台、索引范围和检索日期完整
- [ ] 完整逐字检索式、字段和限制可获得
- [ ] 原始、筛选、去重和最终记录数一致
- [ ] 导出格式、字段、批次和合并过程清楚
- [ ] 作者、机构、国家和关键词清洗留痕
- [ ] 被引参考文献匹配规则清楚
- [ ] 软件、包、插件和版本完整
- [ ] 计数、阈值、归一化、布局和聚类参数完整
- [ ] 自动标签来源及人工改名留痕
- [ ] 节点/边表或等价派生数据可获得
- [ ] 代码、项目文件或操作日志可获得
- [ ] 随机种子及随机算法状态记录（适用时）
- [ ] 图表可由底层数据和参数重建
- [ ] 参数敏感性和代表文献核验已报告
- [ ] 数据许可限制及可行替代复现路径已说明
