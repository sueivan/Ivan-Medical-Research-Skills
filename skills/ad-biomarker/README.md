# Alzheimer Disease Biomarker

Status: **operational domain skill (v0.1.0)**

`ad-biomarker` evaluates AD biomarkers across intended use, disease stage, biological domain, modality, assay/platform, threshold, validation, clinical utility, regulation, and real-world implementation.

## Main outputs

- biomarker research brief
- intended-use and maturity classification
- analytical/clinical validation matrix
- threshold and performance audit
- clinical-utility and implementation assessment
- claim-boundary log

## Invoke

```text
Use @ad-biomarker to evaluate [biomarker] for [intended use] in [population/stage]. Assess specimen/platform, preanalytics, analytical validity, reference standard, threshold, discrimination, calibration, external validation, utility, regulation, availability, and equity.
```

Current guideline, regulatory, reimbursement, and availability information must be verified live.
