# APA 7 Checker

Status: **operational beta (v0.1.0)**

`apa7-checker` audits APA 7 in-text citations, reference lists, DOI/URL presentation, source-type formatting, ordering, quotations, non-English works, and manuscript-wide correspondence.

## Critical separation

```text
reference-verifier = Is this the correct real source and metadata?
apa7-checker = Is the verified source cited and formatted correctly in APA 7?
```

## Invoke

```text
Use @apa7-checker to audit this manuscript and reference list. Verify or flag identities first, build a citation–reference crosswalk, correct APA 7 by source type, and preserve an issue log showing every change.
```
