# APA 7 Checker Quality Checklist

## Target and identity
- [ ] APA 7 requirement is confirmed
- [ ] Current journal/institution deviations are recorded
- [ ] Original manuscript and references are preserved
- [ ] Bibliographic identity is verified or visibly unresolved
- [ ] Publication/retraction status is handled outside formatting

## Crosswalk
- [ ] Every in-text citation matches one reference
- [ ] Every reference is cited in text/table/figure/appendix/supplement or documented
- [ ] Author spellings and years agree
- [ ] Same-author/same-year suffixes agree
- [ ] Duplicates and versions are resolved or flagged

## In-text citations
- [ ] One-, two-, and three-plus-author rules are correct
- [ ] Narrative and parenthetical forms are correct
- [ ] `&`, “and,” commas, and parentheses are correct
- [ ] Multiple works are alphabetized and separated correctly
- [ ] Ambiguous et al. citations are disambiguated
- [ ] Direct quotations include verified locators

## Reference entries
- [ ] Source type is correct
- [ ] Verified author order and initials are preserved
- [ ] Dates match source type
- [ ] Sentence/title case is correctly applied
- [ ] Italicized elements are correct
- [ ] Journal volume/issue and pages/article number are correct
- [ ] DOI is a canonical hyperlink
- [ ] URL and retrieval date rules are appropriate
- [ ] Non-English titles and translations are accurate
- [ ] Reference ordering and hanging indent are correct

## Completion
- [ ] APA errors are separated from metadata and claim-support errors
- [ ] No metadata, page number, translation, or source type is invented
- [ ] Original and corrected forms are traceable
- [ ] Unresolved records are listed for manual review
