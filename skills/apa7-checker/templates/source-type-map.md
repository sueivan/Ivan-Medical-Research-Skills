# APA 7 Source-Type Map

Use only after metadata verification.

| Source type | Core elements to verify | Common error |
|---|---|---|
| Journal article | Authors, year, article title, journal, volume, issue, pages/article number, DOI | Article number forced into pages |
| Online-first article | Authors, date/status, title, journal, DOI | Invented volume/pages |
| Authored book | Authors, year, title, edition, publisher, DOI/URL | Publisher location added |
| Edited-book chapter | Chapter authors/title, editors, book title, pages, publisher, DOI | Book authorship confused with chapter authorship |
| Report/guideline | Author/group, date/version, title, report number, publisher, DOI/URL | Group author/publisher duplicated |
| Webpage | Author, full date, page title, site, URL | Treated as journal article |
| Dataset | Creator, date/version, title, repository/publisher, DOI/URL | No version |
| Software | Author/organization, date/version, title, descriptor, publisher, URL | Tool name cited without version |
| Thesis/dissertation | Author, year, title, type, institution/database/repository, URL | Journal template used |
| Conference item | Authors, date, title, presentation/proceeding type, conference/source, DOI/URL | Abstract and full paper conflated |
| Preprint | Authors, year, title, preprint label/repository, DOI/URL | Presented as peer reviewed |
| Non-English work | Verified original metadata, translation/transliteration as required | Invented English title |
