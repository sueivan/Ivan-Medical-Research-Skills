# Medical Education

Status: **operational beta (v0.1.0)**

`medical-education` supports rigorous curriculum reform and education research across medicine, nursing, pharmacy, public health, allied health, and continuing professional development.

## Main outputs

- educational research brief
- theory-of-change and logic model
- aim–design–outcome matrix
- curriculum/intervention specification
- assessment blueprint and instrument-validation plan
- implementation and fidelity plan
- analysis, ethics, data-governance, and reporting plan

## Invoke

```text
Use @medical-education to design and audit this education study. Define the educational problem, theory, intervention, comparator, unit of allocation and analysis, outcome hierarchy, instrument validity, implementation, statistics, ethics, and inference limits.
```

Satisfaction and self-confidence must not be treated as demonstrated competence.
