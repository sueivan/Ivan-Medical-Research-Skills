# 论断语言校准

## 证据层级与措辞

| 状态 | 推荐措辞 | 避免措辞 |
|---|---|---|
| 直接支持 | 显示、观察到、估计为、与…一致 | 证明、必然、确定 |
| 间接支持 | 可能与…有关、提供支持、提示 | 揭示机制、导致 |
| 探索性 | 生成假设、需验证、可能的解释 | 关键发现、确认 |
| 不精确 | 结果与多种效应相容、不能排除 | 没有差异、完全无效 |
| 不支持 | 本研究不能判断、证据不足以支持 | 基于想象继续下结论 |

## 因果措辞

只有当设计、目标估计量、偏倚控制和假设足以支持时，才使用因果语言。生物学合理性、时间顺序或统计调整单独都不能证明因果。

观察性替代措辞：

- “X 与 Y 相关”，而不是“X 导致 Y”。
- “在所测混杂因素调整后仍存在关联”，而不是“独立危险因素”。
- 若使用“独立相关”，明确独立于哪些模型协变量，不暗示无未测混杂。

## 不显著结果

- CI 同时包含重要获益和伤害：证据不精确。
- CI 排除预设的重要差异：可讨论“未发现临床重要差异”，但需设计充分。
- 非劣效/等效结论必须来自预设界值和适当设计，不能由 `P > 0.05` 推出。

## 推荐强度

- “支持进一步验证”弱于“支持临床应用”。
- “可能有助于风险分层”不等于“应纳入指南”。
- 低确定性证据宜使用“可能”；极低确定性强调“非常不确定”。

## 删除或降级的高风险词

首次、首创、突破性、颠覆性、证实、明确机制、精准预测、显著改善、临床可用、填补空白、未来热点。保留时必须有明确定义和证据。
