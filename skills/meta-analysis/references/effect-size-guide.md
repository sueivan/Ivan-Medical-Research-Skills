# 效应量选择与计算审计

## 二分类结局

- RR：直观但依赖事件定义；先固定事件是获益还是伤害。
- OR：病例对照及部分模型常用；事件常见时不可当作 RR。
- RD：提供绝对差异，但跨基线风险的稳定性可能较差。
- NNT/NNH 通常由相对效应和明确基线风险推导，不宜直接作为主要合并量。
- 稀有事件与单/双零事件需专门方法。连续性校正可能引入偏倚，报告方法并做敏感性分析。

## 连续结局

- MD：相同量表和单位，保留临床单位。
- SMD：概念相同但量表不同；方向必须统一，并说明标准化基准。量表信度和研究间 SD 差异会影响结果。
- 终点值与变化值的可合并性需结合量表和模型判断；缺失变化 SD 时不得无依据猜测相关系数。

## 生存、率和相关

- HR 与时间过程相关，不等于 RR；优先使用正确模型输出的 log(HR) 与 SE。
- 率比需有事件数与人时，区分复发事件和首次事件。
- 相关系数常先作 Fisher z 变换；核对调整/未调整估计及变量方向。

## 通用检查

1. 每项研究是否估计同一问题与时间点？
2. 数值方向是否统一？
3. 比率效应是否在 log 尺度建模？
4. SE、CI、P 值转换是否一致并可回算？
5. 调整估计是否使用相近协变量集？
6. 聚类、配对、交叉设计的方差是否正确？
7. 多臂共享对照是否避免重复计权？

所有转换记录原值、公式、假设、代码和复核人。
