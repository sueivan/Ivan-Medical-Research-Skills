# GRADE 审计清单

- [ ] 每行是明确比较下的单一结局和时间点
- [ ] 患者重要结局在方案阶段选择，包括无数据结局
- [ ] 记录 GRADE 路径、起点和指南版本
- [ ] 风险偏倚依据设计相应工具且结局特异
- [ ] 不一致性不只依据 `I²`
- [ ] 间接性逐项比较 PICO/PECO、时间和情境
- [ ] 不精确性围绕决策阈值而非仅无效线
- [ ] 发表偏倚不只依据漏斗图或 Egger 检验
- [ ] 升级域适用于当前路径且未重复计算
- [ ] 二分类结局报告相对和绝对效应
- [ ] 基线风险来源和计算可追溯
- [ ] 每次降级有具体脚注和证据位置
- [ ] 未把确定性直接转换为推荐强度
- [ ] 第二评审者完成复核并记录分歧
