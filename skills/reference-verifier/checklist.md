# Reference Verifier Quality Checklist

## Input control
- [ ] Every reference has a stable row ID
- [ ] Original citation text is preserved
- [ ] Identifier types are separated
- [ ] Verification date is recorded

## Identity
- [ ] PMID resolves to the intended work
- [ ] DOI resolves to the intended work
- [ ] PMID and DOI correspondence is checked
- [ ] Identifier-free matches use title plus corroborating fields
- [ ] Verification state is assigned explicitly

## Metadata
- [ ] Title is checked
- [ ] Authors are checked
- [ ] Journal and abbreviation are checked
- [ ] Electronic/print years are reconciled
- [ ] Volume, issue, pages, or article number are checked
- [ ] Original and corrected values remain traceable

## Status and duplicates
- [ ] Corrections and errata are checked
- [ ] Retractions, withdrawals, and expressions of concern are checked
- [ ] Exact duplicates are detected
- [ ] Preprint/article and abstract/article relationships are classified

## Integrity and completion
- [ ] Nothing is invented
- [ ] Unverified records are visibly labeled
- [ ] Source conflicts are reported
- [ ] Claim support is not inferred from metadata
- [ ] Formatting occurs only after verification
- [ ] Unresolved records remain in the report
