# CiteSpace Expert

Status: **operational beta (v0.1.0)**

`citespace-expert` designs, audits, interprets, and reports reproducible CiteSpace bibliometric analyses.

## Main outputs

- corpus/import audit
- CiteSpace parameter protocol
- entity-normalization dictionary
- network, cluster, and burst tables
- parameter-sensitivity log
- publication-ready figure specifications and legends
- bounded methods and results text

## Invoke

```text
Use @citespace-expert to design and audit a CiteSpace analysis of this Web of Science corpus. Specify time slicing, node types, selection criterion, pruning, clustering, labeling, bursts, sensitivity analyses, and reproducible reporting.
```

Use `wos-search-builder` first when corpus construction is unresolved.
