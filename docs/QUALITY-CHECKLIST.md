# Repository Quality Checklist

## Scientific integrity
- [ ] No fabricated evidence
- [ ] No unsupported clinical claims
- [ ] Uncertainty is explicit
- [ ] References are verifiable

## Reproducibility
- [ ] Inputs are defined
- [ ] Parameters are reportable
- [ ] Outputs can be reconstructed
- [ ] Deviations are documented

## Skill design
- [ ] Trigger description is precise
- [ ] Scope is bounded
- [ ] Failure modes are addressed
- [ ] Completion criteria are objective

## Maintenance
- [ ] Version is recorded
- [ ] Examples are current
- [ ] Templates match the skill
- [ ] Links and standards are reviewed periodically
